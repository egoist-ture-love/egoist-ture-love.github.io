如何在15分钟内熟练掌握Markdown所有语法

2020年03月18日 685 字 大概 3 分钟

一.认识Markdown

Markdown是一种轻量级标记语言，创始人为约翰·格鲁伯（英语：John Gruber）。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML（或者HTML）文档。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。—维基百科

看完是不是有点懵，用民间的说法就是其实markdown就是一个文本编辑器，让人可以不依赖鼠标就可以编写出你想要格式的文本编辑器。

二.基础语法

##1. 标题

Markdown 的标题只需要在文本前面加上#就可以了，几级标题加几个＃,最多支持6级标题.

#一级标题
##二级标题
###三级标题
####四级标题
#####五级标题
######六级标题

2.列表

Markdown支持两种列表 有序列表 和 无序列表

有序列表—就在文字前面加上编号和.就行了

例如:
  1. 有序列表1
  2. 有序列表2
  3. 有序列表3

无序列表 —文字前面加上 “ - “ 就ok了

  • 无序列表1
  • 无序列表2
  • 无序列表3

3.斜体和加粗

斜体

文字左右各一个” * “ 就完成了 例如: * 斜体 *

斜体

粗体

文字左右各两个” * “就完成了 例如: ** 粗体 **

粗体
##4. 引用

我们可能需要在文章中引用别人的文章,句子,代码…….这个标签就尤为重要了

我们只需要在前面加上 “ > “就完成了.

这个是效果

5. 代码块

对于程序员来说,能用Markdown书写代码尤为重要(嘻嘻~ 毕竟创作之初就是用来书写代码的嘛)

那么格式是:

``` key
代码块
```

例如:

language key
C++ cpp
CSS css
Django django
HTTP http
Java java
XML,HTML xml

6.分割线

三个或者三个以上的 - 或者 * 都可以。

这个是效果

7.插入图片

为了使文章更有说服力，很多人会使用图片来加强理解

![图片alt](图片地址 ‘’图片title’’)
图片alt:图片下面的文字
图片地址：就是很单纯的图片地址
图片title： 鼠标移动到图片上面时候显示的东西

！演示图片

8.表格

上面的语言和关键字就是用表格实现的

呈现效果就是这样的：
name | 111 | 222 | 333 | 444

  • | :-: | :-: | :-: | -:
    aaa | bbb | ccc | ddd | eee|
    fff | ggg| hhh | iii | 000|
